In order to install the necessary packages to run the project code, please run either of the following
commands in the relevant conda or pip prompt in this directory:

conda install --yes --file requirements.txt

or

$ pip install -r requirements.txt

Then install pytorch on windows with:

conda install pytorch torchvision torchaudio cpuonly -c pytorch

or

pip3 install torch==1.8.1+cpu torchvision==0.9.1+cpu torchaudio===0.8.1 -f https://download.pytorch.org/whl/torch_stable.html

or go to https://pytorch.org/ for mac and linux installations.
